üêõ Common Issues & Solutions

Issue: "Failed to fetch"

Cause: Server not running or wrong URL

Solution:

  1. Ensure server is running (see Step 2)
  2. Access admin panel via http://localhost:3000/admin.html
  3. Don't open admin.html directly from file explorer

Issue: "Invalid admin key"

Solution: Use the correct admin key: IEEE2025ADMIN

Issue: "Port 3000 already in use"

Solution: Kill existing Node.js processes:

  1. Open Command Prompt as Administrator
  2. Run: taskkill /f /im node.exe
  3. Restart the server

  3. Restart the server
